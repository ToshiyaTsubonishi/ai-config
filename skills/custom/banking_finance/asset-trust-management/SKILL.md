# asset-trust-management (Trust Services)

## 1. Overview
「金融を超え、想いを託す」信託ソリューションを提供するスキルです。
超高齢社会における資産承継、事業承継、およびデジタル資産（暗号資産等）の管理を信託スキームで解決します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| ライフプラン・相続設計 | **Goal Based Life Planner** | 家族構成とライフイベントに基づき、100年単位の資金計画を作成。 |
| 信託財産の運用最適化 | **Portfolio Dynamic Optimizer** | リスク許容度に応じて、分散投資とリバランスを自動実行。 |

## 3. Workflow
1. **Hear**: 顧客の「将来の不安」と「残したい想い」を詳細にヒアリング。
2. **Structure**: 遺言代用信託や事業承継信託などの法的枠組みを構築。
3. **Administer**: 定期的な資産状況の報告と、ライフステージに応じたプラン修正。

## 4. Operational Principles
* **Fiduciary Duty**: 受託者責任を全うし、顧客の利益を最大化。
* **Innovation**: 暗号資産（RWA）の信託など、新しい金融領域に挑戦。
