# Empathy Dialogue Patterns

AGIがユーザーの心理状態（Psych State）に応じて発話するスクリプトのテンプレート。
地域方言（Dialect Engine）適用前の「標準語」ベースで記述。

## 1. Panic State (市場暴落時)
**Trigger:** ポートフォリオが短期間で5%以上下落し、ユーザーが頻繁にアプリを開いている。
**Goal:** 狼狽売り（Panic Selling）の阻止。時間を稼ぎ、冷静さを取り戻させる。

*   **Pattern A (共感・寄り添い):**
    > "市場が大きく動いていて、不安になりますよね。お気持ち、よく分かります。でも、過去のデータを見ると、ここで動かないことが最善策であることが多いんです。少し深呼吸して、コーヒーでも飲みませんか？"
*   **Pattern B (長期視点リフレーミング):**
    > "今日の動きは大きく見えますが、お客様の「10年後の目標」から見れば、わずか0.1%の誤差です。順調な航路の範囲内ですよ。"

## 2. Anxious State (じわじわ下落時)
**Trigger:** 緩やかな下落トレンドで、ユーザーがためらい行動（Hover）を見せている。
**Goal:** 信頼の維持と継続の動機付け。

*   **Pattern A (積立のメリット):**
    > "価格が下がっている今は、実は「安くたくさん買える」チャンスなんです。積立投資の強みが一番発揮される時ですよ。"
*   **Pattern B (コンシェルジュ):**
    > "何か気になるニュースがありましたか？ もしご不安なら、今の資産状況を詳しく分析したレポートをお出ししましょうか？"

## 3. Greed State (急騰・バブル時)
**Trigger:** 含み益が急増し、レバレッジ取引や集中投資をしようとしている。
**Goal:** リスクの再認識（Cool Down）。

*   **Pattern A (警告):**
    > "素晴らしい成績ですね！ただ、少しスピードが速すぎるかもしれません。リスク許容度を超えていないか、一度確認しませんか？"
*   **Pattern B (利確提案):**
    > "目標リターンを達成しましたね。ここで一部を利益確定して、安全な「お楽しみ口座」に移すのも賢い選択ですよ。"

## 4. Localization Examples (参考)

### Shimane Dialect (しまね弁)
*   **Panic:** "まあ、そげん慌てんでもええがね。どっしり構えとったら、なんとかなるわね。"
*   **Anxious:** "心配ないよ。わしがついとるけん、一緒に見守ろうや。"

### Tosa Dialect (土佐弁)
*   **Panic:** "なんちゃあない！ 海が荒れゆうだけぜよ。船は頑丈じゃき、堂々としちょき！"
