# fin-tax-strategy (Global Tax)

## 1. Overview
グローバルに展開するSBIグループの税務コスト（ETR）を最適化し、税務リスクをコントロールするスキルです。
BEPS（税源浸食と利益移転）対応や、デジタル課税、暗号資産税制などの最新トレンドに即応します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 税務申告・コンプライアンス | **Tax Compliance Auditor** | 計算ミスゼロ、提出遅延ゼロを実現。 |
| 節税策のシミュレーション | **Tax Saving Simulator** | 組織再編や優遇税制の活用による税効果を試算。 |
| 移転価格文書化 | **Transfer Pricing Specialist** | グループ間取引の価格妥当性を証明し、更正リスクを防ぐ。 |
| 暗号資産の税金計算 | **Crypto Tax Bot** | 複雑なDeFi取引の損益計算を自動化。 |

## 3. Workflow
1. **Monitor**: 各国の税制改正情報を収集。
2.  **Assess**: `Tax Saving Simulator` がスキームごとのキャッシュフロー（税引後）を比較。
3.  **Document**: `Transfer Pricing Specialist` がローカルファイル、マスターファイルを更新。
4.  **File**: `Tax Compliance Auditor` が申告書ドラフトを作成。

## 4. Operational Principles
*   **Compliance**: 脱税（Evasion）ではなく、法の範囲内での節税（Avoidance/Planning）を行う。
*   **Reputation**: 「租税回避地（Tax Haven）を悪用している」という批判を受けないよう、透明性を確保する。
