# media-content-production (Creative Studio)

## 1. Overview
生成AIを駆使して、マーケティングに必要なクリエイティブ素材を「爆速」で量産するスキルです。
人間のクリエイターは「ディレクション（指示）」に集中し、制作作業はAIが担うことで、圧倒的なコストダウンとスピードを実現します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| クリエイティブ自動生成・管理 | **Creative Automation Bot** | 台本、画像、音声を一括生成し、動画やバナーとして統合。 |

## 3. Workflow
1. **Brief**: マーケターがキャンペーンの目的とターゲットを入力。
2.  **Generate**: `Creative Automation Bot` が数十パターンのバリエーションを作成。
3.  **Review**: ブランドガイドライン（色、フォント、禁止用語）に適合しているか自動チェック。
4.  **Export**: 承認された素材を、各広告媒体の仕様に合わせて書き出し。

## 4. Operational Principles
*   **Consistency**: ブランドイメージを損なわないよう、トーン＆マナーを統一する。
*   **Speed**: トレンドが発生してから数時間以内にコンテンツを公開する。
