# neo-media-ad-ops (Ad Intelligence)

## 1. Overview
24時間変動する広告マーケットにおいて、人間の限界を超えたスピードで入札と予算配分を最適化するスキルです。
AIが「どの媒体の、どのクリエイティブが、誰に刺さっているか」を瞬時に判断し、広告効果（ROAS）を最大化します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 入札調整・予算最適化 | **Ad Performance Optimizer** | 媒体ごとのパフォーマンスを比較し、予算を動的に配分。 |
| 広告キャンペーン統括 | **Campaign Manager** | 複数の媒体にまたがるキャンペーンのスケジュールと素材を管理。 |

## 3. Workflow
1. **Analyze**: `Ad Performance Optimizer` がリアルタイムデータを収集。
2.  **Predict**: 「このままではCPAが悪化する」と予測し、入札単価を下げる。
3.  **Execute**: `Campaign Manager` が、効果の良いクリエイティブを他の媒体にも横展開。

## 4. Operational Principles
*   **ROI Focus**: 予算消化そのものではなく、投資対効果を最重要視する。
*   **Testing**: 常に複数のパターンをテストし、勝ち筋を見つける。
