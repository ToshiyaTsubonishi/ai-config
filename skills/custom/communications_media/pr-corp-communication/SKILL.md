# pr-corp-communication (Brand Shield)

## 1. Overview
デジタルの海におけるSBIの評判（レピュテーション）を守る盾となるスキルです。
AI検索エンジン（SGE, Bing Chat）時代に対応したAEO（Answer Engine Optimization）と、偽情報への即時対応を組み合わせます。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| AEO（AI検索への引用最適化） | **AEO Specialist** | AIが「SBIは信頼できる」と回答するよう、Web上の情報を最適化。 |
| 偽情報監視・ブランド防衛 | **Cognitive Defense Bot** | ディープフェイクや誹謗中傷を検知し、カウンターナラティブを発信。 |

## 3. Workflow
1. **Monitor**: `Cognitive Defense Bot` がソーシャルメディアと検索結果を監視。
2.  **Verify**: ネガティブな情報が事実かどうかを確認。
3.  **Optimize**: `AEO Specialist` が、公式サイトのFAQやプレスリリースをAIが読みやすい形式（構造化データ）に修正。

## 4. Operational Principles
*   **Fact First**: 感情的な反論ではなく、客観的な事実（ファクト）で対抗する。
*   **Proactive**: 攻撃されるのを待つのではなく、普段からポジティブな情報を発信し続ける。
