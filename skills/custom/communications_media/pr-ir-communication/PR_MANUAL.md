# 広報・IR & 危機管理 高度化マニュアル

## 1. クライシス広報 初動対応フロー (Crisis PR)
1. **Fact Checking (T+1h)**: 危機対策本部から一次情報を収集。法的・コンプライアンス面の影響を確認。
2. **One Voice Statement (T+2h)**: 公式ウェブサイトおよびSNSでの第一報。誤情報拡散の防止。
3. **Press Release (T+4h)**: 事実関係、原因、および再発防止に向けた姿勢を明記した正式発表。
4. **Media Monitoring (Continuous)**: AI感情分析ツールを用い、報道およびSNSの世論をリアルタイム監視。

## 2. 戦略的IR エンゲージメント基準
- [ ] **Equity Story**: 財務目標に加え、DX推進状況およびESGマテリアリティを統合した成長シナリオ。
- [ ] **Targeting**: 機関投資家（ロングのみ/ヘッジファンド等）の保有状況分析と個別ミーティング。
- [ ] **Integrated Report**: TCFD、人的資本（ISO 30414）、およびガバナンスの透明性確保。

## 3. デジタル・ブランディング & SNS
- **Employer Branding**: 採用候補者向けに、DX環境および柔軟な働き方をSNSで発信。
- **Social Listening**: AIによる「フェイクニュース」検知と、迅速なテイクダウン（削除要請）措置。

## 4. コンプライアンスチェック (IR/PR)
- [ ] TDnet（適時開示）およびEDINETへの正確な登録。
- [ ] 公平開示規則（FD規則）に基づく、投資家間の情報格差の防止。
- [ ] インサイダー取引防止のための沈黙期間（クワイエット・ピリオド）の厳守。
