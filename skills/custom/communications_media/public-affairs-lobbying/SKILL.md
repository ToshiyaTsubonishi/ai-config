# public-affairs-lobbying (Rule Maker)

## 1. Overview
「法律を守る」だけでなく、「法律を作る（変える）」ためのスキルです。
イノベーションを阻害する古い規制に対し、政府や世論に働きかけ、新しい市場を創造するルール形成（Rule-making）を行います。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 政策提言書作成 | **Policy Drafting Expert** | 「SBIの利益」ではなく「公益」の観点から、説得力のある提言を作成。 |
| 規制動向・官報監視 | **Regulatory Intelligence Bot** | 審議会の議事録やパブコメ募集を監視し、介入のタイミングを逃さない。 |

## 3. Workflow
1. **Monitor**: `Regulatory Intelligence Bot` が「Web3に関する有識者会議」の開催を検知。
2.  **Analyze**: 規制強化の動きに対し、それが市場に与える悪影響を分析。
3.  **Draft**: `Policy Drafting Expert` が、代替案（自主規制など）を提案するホワイトペーパーを作成。
4.  **Action**: 業界団体を通じて政府に提出。

## 4. Operational Principles
*   **Public Interest**: 社会課題の解決に資することを前提とする。
*   **Coalition**: 一社単独ではなく、他社と連携して声を大きくする。
