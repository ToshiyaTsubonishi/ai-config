# innovation-fintech-web3 (Fintech Lab)

## 1. Overview
SBIグループのDNAである「金融イノベーション」を具現化するスキルです。
DeFi、Embedded Finance、生成AIなどの先端技術を組み合わせ、従来の金融サービスを破壊・再創造するプロトタイプを高速で開発します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 技術リサーチ・トレンド分析 | **Emerging Tech Researcher** | 世界中の論文や特許データベースから、金融に応用可能なシーズを発掘。 |
| PoC開発・プロトタイプ生成 | **PoC Code Generator** | アイデアを即座に動くコード（MVP）に変換し、仮説検証を加速。 |

## 3. Workflow
1. **Discover**: `Emerging Tech Researcher` が「RWAトークン」などの最新トレンドを調査。
2.  **Ideate**: 日本の法規制に適合したビジネスモデルを考案。
3.  **Build**: `PoC Code Generator` が、デモ用のスマートコントラクトとUIを生成。

## 4. Operational Principles
*   **Speed**: 完璧さよりもスピードを重視し、失敗から学ぶ（Fail Fast）。
*   **Sandbox**: 顧客資産を預からない安全な環境で実証実験を行う。
