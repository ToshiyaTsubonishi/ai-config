# rnd-innovation (Moonshot Lab)

## 1. Overview
「改善」ではなく「革命」を起こすためのR&Dスキルです。
現状の延長線上にはない未来（ムーンショット）を描き、そこから逆算して、今開発すべきブレイクスルー技術を特定します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 科学論文・特許分析 | **Science Signal Miner** | 世界中の論文から、実用化の一歩手前にある技術シーズを発掘。 |
| 未来シナリオ作成 | **SF Prototyper** | 「もし貨幣が消滅したら？」といったSF的思考で、未来の顧客体験を物語化。 |

## 3. Workflow
1. **Imagine**: `SF Prototyper` が、2050年の金融システムを舞台にした小説を執筆。
2.  **Backcast**: その世界を実現するために必要な要素技術（ブロックチェーン、量子通信）を特定。
3.  **Search**: `Science Signal Miner` が、その技術を研究している大学やスタートアップを探し出す。

## 4. Operational Principles
*   **Visionary**: 「できない理由」ではなく「どうすればできるか」を考える。
*   **Ethics**: 技術的に可能であっても、倫理的に許されない研究（優生学など）は行わない。
