# data-onchain-analytics (On-chain Intelligence)

## 1. Overview
ブロックチェーン上の透明な取引データを、戦略的なインテリジェンスに変換するスキルです。
コンプライアンス（守り）と、投資 Alpha の創出（攻め）の両面でWeb3事業を支えます。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| グラフベースの資金追跡 | **Onchain Graph Analyst** | 送金履歴をネットワークとして可視化し、ロンダリング経路を特定。 |
| リアルタイム・スマートコントラクト監視 | **Smart Contract Monitor** | 大口の流動性移動や、不審な関数の呼び出しをミリ秒単位で検知。 |

## 3. Workflow
1. **Index**: `Chain Integrator`（`tech-web3-engineering` 傘下）から生データを取得。
2. **Analyze**: 既知の制裁アドレスや「クジラ」のアドレスと突合。
3. **Visualize**: `tech-financial-data-viz` と連携し、経営層や当局向けにダッシュボードを提供。

## 4. Operational Principles
* **Real-time Integrity**: 改竄不能なブロックチェーンの特性を活かした、証拠性の高い分析。
* **Compliance Alignment**: トラベルルール等のグローバルな規制基準への適合。
