# data-strategy-science (Data Wealth)

## 1. Overview
「データは新しい原油」であり「SBIの富」であるという認識のもと、グループ内のデータを収益化可能な「資産」へ昇華させるスキルです。
MLOpsによるAI開発の高速化と、顧客体験の超パーソナライゼーションを実現します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| データ資産評価・トークン化 | **Data Asset Tokenizer** | データの利用権をRWAとして評価し、社内でのデータ流通を促進。 |
| ハイパー・パーソナライゼーション | **Hyper-Personalization Strategist** | ライフステージに合わせた「今、この瞬間の最適提案」を算出。 |
| MLOps基盤構築・運用 | **MLOps Orchestrator** | AIモデルの劣化（ドリフト）を検知し、自律的な再学習ループを構築。 |

## 3. Workflow
1. **Unify**: `identity-orchestrator-sbi` 連携により、分散したデータを「SBI ID」で名寄せ。
2. **Train**: `Model Factory Agent`（`tech-datarobot-integration` 傘下）を用いて、最適な予測モデルを自動生成。
3. **Deploy**: リアルタイム推論基盤を通じて、フロントエンド（`tech-frontend-engineering`）に結果をフィードバック。

## 4. Operational Principles
* **Privacy First**: データの利活用とプライバシー保護のトレードオフを解消。
* **Assetization**: 「溜めるだけ」のデータから「稼ぐ」データへの変革。
