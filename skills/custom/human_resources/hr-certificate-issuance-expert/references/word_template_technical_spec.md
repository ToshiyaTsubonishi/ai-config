# 人事証明書 Wordテンプレート・ブックマーク定義書

## 1. テンプレート設計の原則
差し込みデータの長さ（例：長い住所や複数の職務内容）に依存せず、常に1ページ以内に美しく収めるための設計指針。

### 1.1 ブックマーク・フィールド命名規則
AIがデータの流し込み位置を特定するための標準ID。

| 論理名 | ブックマーク/フィールド名 | 最大文字数目安 | 溢れた時の処理 |
| :--- | :--- | :--- | :--- |
| 社員氏名 | `BM_EmployeeName` | 20文字 | フォントサイズを自動縮小(AutoFit) |
| 現住所 | `BM_CurrentAddress` | 100文字 | 2行まで許容、それ以上は3行目へ |
| 役職名 | `BM_PositionTitle` | 30文字 | 略称の使用を検討（AI提案） |
| 年収額 | `BM_AnnualSalary` | 15文字 | 3桁区切りカンマ必須 |

## 2. レイアウト崩れ防止策 (Technical Measures)
- **表形式の活用**: 項目名と内容を独立したセルに配置し、セル高さを「固定」に設定する。
- **テキストボックスの禁止**: 描画オブジェクトは位置がずれやすいため、本文内の「枠線なし表」で代用する。
- **改行コードの正規化**: 外部システムから取得したデータに含まれる不要な `
` や `` は、差し込み前にAIエージェントがスペースへ置換する。

## 3. 印刷・出力トラブル対応
- **印影（デジタル印）の重なり**: 印影画像を「前面」に配置し、透過PNGを使用。背景文字が隠れないよう座標をミリ単位で指定。
- **フォント埋め込み**: PDF変換時、全フォントを埋め込む設定をデフォルトとする（Mac/Windows間の互換性確保）。
