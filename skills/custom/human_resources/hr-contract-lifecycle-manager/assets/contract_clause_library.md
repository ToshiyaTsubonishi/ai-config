# 雇用契約・変更合意 特約条項ライブラリ (2026年版)

## 1. 概要
標準テンプレートでは対応できない個別合意事項について、法的有効性を保ちつつ柔軟に記載するための文面集。

## 2. カテゴリ別条項例

### 2.1 就業場所・業務変更の範囲 (2024改正義務化対応)
- **文面例**: 「就業場所は、本社（東京都港区）および会社が指定するテレワーク実施場所とする。ただし、業務上の必要がある場合は、会社が設置する国内外の他の事業所への異動を命じることがある。」
- **AI判定基準**: 変更の範囲が「具体的」かつ「限定的」であるかを確認。

### 2.2 試用期間の延長
- **文面例**: 「当初の試用期間（3ヶ月）において、勤務成績または能力が本採用基準に達しないと判断される場合、会社は最大3ヶ月を限度として試用期間を延長することができる。」
- **リスク注記**: 無制限の延長は無効となるリスクがあるため、上限期間の有無をチェック。

### 2.3 競業避止義務 (退職後)
- **文面例**: 「乙は退職後1年間、甲の営業秘密を不当に利用し、甲と競合する事業を営む他社へ就職し、または同様の事業を自ら営んではならない。」
- **AI判定基準**: 期間（1年以内推奨）および地理的範囲の妥当性を検証。

## 3. 変更合意書 (覚書) 構成案
1. **原契約の特定**: YYYY/MM/DD付 雇用契約書の第X条を以下の通り変更する。
2. **変更後の条項**: (上記のライブラリから引用)
3. **効力発生日**: 変更内容がいつから適用されるかを明記。
4. **不利益変更の同意**: 賃金引き下げ等の場合は、本人の「自由な意思に基づく同意」を示す文言を必須とする。
