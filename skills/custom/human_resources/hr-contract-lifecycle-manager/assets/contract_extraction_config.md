# 電子契約 抽出項目・クラウドサイン連携定義

## 1. 契約書AI抽出項目 (Mapping)
Word/PDFの契約書からAIが抽出・管理表へ転記する項目。

| 項目 | 抽出対象文字列 (例) | 転記先フィールド |
| :--- | :--- | :--- |
| 契約種別 | 雇用契約書、労働条件通知書、覚書 | `ContractType` |
| 氏名 | [氏名] (甲)、[氏名] (乙) | `EmployeeName` |
| 住所 | [住所] | `Address` |
| 契約開始日 | YYYY年MM月DD日、令和XX年... | `StartDate` |
| 契約終了日 | 定めなし、YYYY年MM月... | `EndDate` |
| 基本給 | 月給 XXX,XXX 円 | `BaseSalary` |
| 職務内容 | 別紙の通り、...の業務 | `JobDescription` |

## 2. クラウドサイン連携用 CSVテンプレート構造
電子締結の一括送信に使用するCSVヘッダー項目。

```csv
"書類名","宛先名","宛先メールアドレス","共有先メールアドレス","管理番号","備考"
"雇用契約書_田中太郎","田中太郎","tanaka@example.com","hr-admin@example.com","EMP-2025-001","2025/4/1入社"
```
