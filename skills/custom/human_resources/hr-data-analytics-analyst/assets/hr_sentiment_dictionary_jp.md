# 日本語人事感情分析辞書 (Sentiment Dictionary)

## 1. 目的
従業員アンケート、口コミ、1on1記録の自由記述欄から、離職予兆や組織課題をAIが高精度に抽出するための極性定義。

## 2. ネガティブ・キーワード & 極性強度 (Scope: HR Context)
数値は極性（-1.0: 強い不満 〜 0: 中立）。

| キーワード | 極性 | HR的文脈の解釈 |
| :--- | :--- | :--- |
| **納得できない** | -0.8 | 評価、報酬、または意思決定プロセスへの強い不信感。 |
| **不当な扱い** | -0.9 | ハラスメントや差別、または過度な業務偏重の懸念。 |
| **キャリアが見えない**| -0.7 | 成長機会の欠如、リスキリング環境への不満。 |
| **疲労 / 限界** | -0.8 | メンタルヘルス不調、または過重労働の危険信号。 |
| **古い体質** | -0.6 | DX推進の阻害要因、硬直化した組織文化。 |
| **相談相手がいない**| -0.7 | 心理的安全性の低下、孤立化の予兆。 |

## 3. 「本音」を汲み取る分析ロジック (Nuance Mapping)
- **逆接の強調**: 「制度は良いが、[不満]」の構造における、後半のネガティブ極性を1.5倍に重み付け。
- **婉曲表現の検知**: 「検討します」「前向きに...」等の表現が特定の文脈で繰り返される場合、停滞・諦めとしてスコアリング。
- **時間の経過**: 同一人物の感情スコアが3ヶ月連続で低下した場合、自動的に `Retention Alert` を発報。
