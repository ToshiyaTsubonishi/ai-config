# 離職予測AIモデル 高度特徴量 (Feature Engineering) カタログ

## 1. モデル構築の目的
過去の離職者・在職者の人事データを学習し、現職社員の「離職確率」をスコアリングすることで、早期の1on1実施やキャリア支援に繋げる。

## 2. 推奨される特徴量項目一覧 (2026年最新)

### 2.1 勤怠・負荷データ
- **直近3ヶ月の残業時間トレンド**: 単純な総時間ではなく「増加傾向」を算出。
- **有給休暇取得率（逆転指標）**: 取得率が急激に低下した場合のリスク。
- **深夜労働回数**: 身体的・精神的負荷の代替指標。

### 2.2 評価・キャリアデータ
- **評価ランクの変動**: 期待値とのギャップ（評価が下がった、または停滞した期間）。
- **昇進・昇格までの平均期間との差**: 「出世の遅れ」を感じている層の特定。
- **研修受講履歴の「停滞」**: 自己啓発を止めた時期は離職の予兆となる。

### 2.3 給与・市場価値データ
- **同業他社平均給与との乖離率**: 外部サイト（口コミ等）の推定市場価値との比較。
- **ボーナス支給額の前年比**: 報酬への納得感。

### 2.4 エンゲージメント・感情データ (非定型)
- **サーベイの自由記述欄の「単語感情スコア」**: 不満、疲弊、諦めなどのキーワード。
- **サーベイ回答までの「時間」**: 回答が極端に早い（適当）、または遅い（迷い）場合の相関。

## 3. 分析手法と留意点
- **手法**: ランダムフォレスト、XGBoost、またはLightGBM等の勾配ブースティング木が有効。
- **倫理的配慮**: 予測結果を「退職させないための監視」ではなく「キャリア支援」のために使うことを明文化し、本人にバイアスがかからないよう注意する。
- **精度検証**: 適合率（Precision）よりも再現率（Recall）を重視し、リスク層を漏らさず検知する。
