# hr-global-mobility (Border Eraser)

## 1. Overview
国境を越えた人材移動をシームレスにするスキルです。
煩雑なビザ手続きや税務処理を自動化し、社員が現地での業務に即座に集中できる環境を整えます。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 赴任生活サポート | **Expat Support Bot** | 現地の生活情報や社内規定に関する質問に24時間対応。 |
| ビザ・税務手続き | **Immigration Specialist** | 各国の移民法や租税条約に基づき、適法な滞在を支援。 |

## 3. Workflow
1. **Initiate**: 人事から赴任辞令が出る。
2.  **Process**: `Immigration Specialist` が必要書類リストを作成し、ビザ申請を開始。
3.  **Onboard**: `Expat Support Bot` が、現地の銀行口座開設や住居探しをガイド。

## 4. Operational Principles
*   **Compliance**: 不法就労リスクをゼロにする。
*   **Well-being**: 異文化環境でのメンタルヘルスケアを重視する。
