# 二次健康診断等給付制度 実務ガイド (2025年版)

## 1. 制度の概要
労働者災害補償保険法に基づき、直近の定期健康診断（一次健康診断）で「脳・心臓疾患の発症に関連する検査項目」に異常の所見が認められた場合に、二次健康診断および特定保健指導を無料で受けられる制度。

## 2. 対象者の判定基準
一次健康診断において、以下の4項目すべてについて「異常の所見」があると診断された労働者。
1.  **血圧検査**: 収縮期140mmHg以上または拡張期90mmHg以上（または薬物治療中）
2.  **血中脂質検査**: 空腹時中性脂肪150mg/dL以上、HDLコレステロール40mg/dL未満、またはLDLコレステロール140mg/dL以上（または薬物治療中）
3.  **血糖検査**: 空腹時血糖110mg/dL以上または随時血糖140mg/dL以上（または薬物治療中）
4.  **腹囲・BMI**: 腹囲が男性85cm以上、女性90cm以上。またはBMIが25以上。

※産業医が個別に「異常の所見」と認める場合も含む。
※既に脳・心臓疾患の症状がある場合は対象外。

## 3. 人事担当者の実務フロー
1.  **対象者の抽出**: 健診機関から届く結果データに基づき、上記4条件を満たす者をリストアップ。
2.  **受診勧奨**: 抽出から1週間以内に本人へ通知（メールおよび書面）。
3.  **書類準備**: 「二次健康診断等給付金請求書（様式第16号の10の2）」の事業主証明欄を記入し本人へ交付。
4.  **受診確認**: 健診日から3ヶ月の期限内に受診したかをトラッキング。
5.  **結果の保存**: 二次健診の結果は5年間保存する義務がある。

## 4. 費用請求の仕組み
- 医療機関が労働者から受け取った請求書（様式第16号の10の2）を労働基準監督署へ提出することで、医療機関に直接費用が支払われる。
- 企業側での立替払いや精算は原則不要。

## 5. 法的根拠
- 労働安全衛生法 第66条（健康診断）
- 労働者災害補償保険法 第26条（二次健康診断等給付）
