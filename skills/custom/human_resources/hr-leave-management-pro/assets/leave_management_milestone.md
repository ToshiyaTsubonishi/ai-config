# 産休・育休・介護休業 手続きマイルストーン (2025年改正対応)

| 時期 | アクション | 担当者 | 備考 |
| :--- | :--- | :--- | :--- |
| 妊娠・介護発生の報告 | 個別の意向確認・面談の実施 | 人事・上長 | 2025年4月より意向確認が義務化 |
| 休業開始1ヶ月前まで | 休業届、社会保険料免除申請の受理 | 従業員・人事 | 産後パパ育休（出生時育児休業）は2週間前まで |
| 休業開始時 | 育児休業給付金受給資格確認票の提出 | 人事 | ハローワークへの届出 |
| 出産後 (産休中) | 出産手当金・出産育児一時金の申請案内 | 人事 | 健康保険組合への手続き |
| 子が3歳になるまで | 柔軟な働き方の選択 (2つ以上の措置) | 従業員・人事 | 2025年改正：テレワーク、時差出勤、短時間勤務等 |
| 復職前 (1ヶ月前) | 復職面談、保育園状況の確認 | 上長・人事 | 復職後のキャリアパス、時短勤務の有無を確認 |
| 復職時 | 社会保険料改定、育児休業等終了届の提出 | 人事 | 給与変動に応じた標準報酬月額の改定 |

## 2025年4月からの重要変更点
- **柔軟な働き方の義務化**: 3歳以上小学校就学前の子を育てる社員に対し、5つの選択肢（始業終業時刻変更、テレワーク、保育支援、休暇付与、短時間勤務）から2つ以上の措置を講じる。
- **子の看護等休暇の拡充**: 対象が小3まで拡大、行事参加等も理由として容認、勤続6ヶ月未満も対象。
