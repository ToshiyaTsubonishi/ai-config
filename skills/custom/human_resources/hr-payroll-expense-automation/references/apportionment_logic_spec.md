# 出向費用 按分計算ロジック・税務処理定義

## 1. 基本計算ロジック (Pseudocode)
出向者の給与負担金を、出向元・出向先間で正確に按分するためのロジック。

```python
# 出向費用按分計算関数
def calculate_secondment_cost(total_salary, base_hours, secondment_ratio):
    """
    total_salary: 総支給額 (給与 + 賞与引当 + 法定福利費)
    base_hours: 規程上の就業時間
    secondment_ratio: 出向先での従事比率 (例: 0.5 = 50%)
    """
    # 1. 出向先負担額の算出
    # 原則として、従事比率に基づき算出。不課税取引として処理。
    target_cost = total_salary * secondment_ratio
    
    # 2. 較差補填金の判定
    # 出向先の給与水準が出向元より低い場合、出向元が差額を負担。
    # この較差補填金は出向元の損金(給与)として計上可能。
    
    return {
        "dispatch_fee": target_cost,  # 出向先へ請求する額 (不課税)
        "parent_burden": total_salary - target_cost # 出向元が負担する額
    }
```

## 2. 消費税および会計処理の定義
- **消費税区分**: **「不課税」**
  - 出向は「労働の対価」であり、事業者が行う「資産の譲渡等」には該当しない。
  - 注意: 「人材派遣」と判定された場合は「課税」となるため、指揮命令系統が出向先にあることを契約書で確認する。
- **勘定科目**:
  - 請求側 (出向元): `雑収入` または `給与手当(マイナス)`
  - 支払側 (出向先): `給与手当` または `支払手数料`

## 3. 非課税経費・非課税通勤費の精算ルール
- 出向先での業務に伴う実費精算（旅費・接待費等）は、出向先が直接負担する。
- 通勤手当については、主たる事業所（通常は出向元）が一括して支払い、按分対象に含める。
