# 採用KPIダッシュボード 定義・分析フレームワーク

## 1. 主要KPI (指標) の定義
| KPI項目 | 計算式・定義 | 2025年 目標値案 |
| :--- | :--- | :--- |
| **応募単価 (CPA)** | 採用コスト計 ÷ 総応募数 | 1.5万円以下 |
| **書類通過率** | 一次選考通過数 ÷ 書類選考対象数 | 30% |
| **内定承諾率** | 内定承諾数 ÷ 内定出し総数 | 70%以上 |
| **辞退率 (内定後)** | 内定辞退数 ÷ 内定出し総数 | 20%以下 |
| **歩留まり (フェーズ別)** | 前フェーズからの遷移率 | 各フェーズで定義 |

## 2. 歩留まり・ボトルネック分析 (Conversion Analysis)
- **書類 → 一次面接**: 通過率が低い場合は「求人票(JD)とターゲットの乖離」を疑う。
- **一次 → 最終面接**: 通過率が低い場合は「一次面接官の目線合わせ不足」を疑う。
- **最終 → 内定**: ここでの辞退が多い場合は「競合他社との条件差」または「魅力付け不足」を特定。

## 3. ダッシュボード構成要素
- リアルタイム進捗（月次・週次）
- チャネル別効果（リクナビ、マイナビ、エージェント、ダイレクトリクルーティング）
- 面接官別 評価傾向（甘辛傾向の可視化）
