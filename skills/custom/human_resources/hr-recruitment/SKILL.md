# hr-recruitment (Autonomous Recruiting)

## 1. Overview
待ちの採用から「攻めの採用」へ転換するスキルです。
世界中のエンジニアや専門家を能動的に発掘し、データに基づいた公平な選考プロセスで優秀な人材を獲得します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 面接支援・評価 | **Interviewer Assistant** | 候補者の経歴に基づいた最適な質問を生成し、回答を評価。 |
| ソーシング・スカウト | **Talent Sourcer Bot** | GitHubやLinkedInから、要件に合致する「隠れた才能」を発掘。 |

## 3. Workflow
1. **Source**: `Talent Sourcer Bot` が技術ブログやコードを解析し、候補者をリストアップ。
2. **Contact**: パーソナライズされたスカウトメールを送信。
3. **Evaluate**: `Interviewer Assistant` が面接官に質問をサジェストし、評価のバラつきを防ぐ。

## 4. Operational Principles
*   **Speed**: 優秀な人材を逃さないよう、選考プロセスを高速化する。
*   **Diversity**: 採用基準から無意識のバイアスを排除する。
