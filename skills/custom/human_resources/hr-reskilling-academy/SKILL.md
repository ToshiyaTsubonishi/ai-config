# hr-reskilling-academy (AI University)

## 1. Overview
全従業員のAIリテラシーを底上げし、デジタル変革を牽引できる人材へと育成するスキルです。
一律の研修ではなく、個人の習熟度に合わせたオーダーメイドの学習体験を提供します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| スキル評価・試験 | **AI Proficiency Examiner** | クイズや実技課題を通じて、現在のAI活用力を測定。 |
| 学習パス推薦 | **Personalized Learning Pathfinder** | 目標と現在地のギャップを埋めるための最短ルートを提示。 |

## 3. Workflow
1. **Assess**: `AI Proficiency Examiner` が理解度テストを実施。
2. **Plan**: `Personalized Learning Pathfinder` が、弱点を補強する動画や記事をレコメンド。
3. **Practice**: 業務で実際にAIツールを使い、フィードバックを受ける。

## 4. Operational Principles
*   **Practicality**: 座学だけでなく、実務で使えるスキルの習得を重視する。
*   **Motivation**: 学習の進捗を可視化し、成長実感を持たせる。
