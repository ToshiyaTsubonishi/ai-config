# HR Tech AI活用ガバナンス・倫理ガイドライン (2026年版)

## 1. 基本原則
人事領域におけるAI（生成AI、予測モデル）の活用は、以下の原則を遵守しなければならない。
1.  **公平性と非差別**: 性別、年齢、国籍等の属性による不当なバイアスを排除する。
2.  **透明性と説明責任**: AIの判断根拠（特徴量等）を説明可能にし、最終判断は人間が行う。
3.  **プライバシー保護**: 個人情報の機微性を鑑み、必要最小限のデータ利用と厳格な権限管理を行う。

## 2. 用途別制限事項
- **採用スクリーニング**: AIの結果のみで不採用を自動決定することを禁止する。必ず人間がスコアを確認するプロセスを設ける。
- **離職予測**: 予測結果を本人に告知する際は、不利益変更（評価の引き下げ等）に繋がらないよう細心の注意を払う。
- **問い合わせ対応**: AIが「回答できない」と判断した事項は、速やかに担当者へ繋ぐエスカレーションパスを維持する。

## 3. セキュリティ・品質管理
- **ハルシネーション（誤情報）対策**: 規程類の回答には必ず「出典（規程名・条文）」を明示させる。
- **シャドーAIの禁止**: 個人用AIアカウント等での機密情報の処理を厳禁とし、承認された「人事専用AI環境」のみを利用する。
- **定期監査**: 半年に一度、AIの回答精度とバイアスの有無を第三者（または担当外のチーム）がチェックする。

## 4. 法令・ガイドライン参照
- 経済産業省・総務省「AI事業者ガイドライン」
- 欧州連合 (EU) AI法 (AI Act)
- 個人情報保護法 (2025/2026改正対応)
