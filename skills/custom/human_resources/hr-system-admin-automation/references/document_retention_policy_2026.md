# 人事・労務書類 法定保存期間 & 廃棄ガイドライン (2026年版)

## 1. 主要書類の保存期間マトリクス
労働基準法および関連法規に基づく定義。

| 書類種別 | 保存期間 | 起算日 | 根拠法 |
| :--- | :--- | :--- | :--- |
| 労働者名簿 | 5年 (当面3年)| 退職・死亡の日 | 労基法109条 |
| 賃金台帳 | 5年 (当面3年)| 最後の記入日 | 労基法109条 |
| 雇入れ・退職書類 | 5年 (当面3年)| 退職・死亡の日 | 労基法109条 |
| **年次有給休暇管理簿**| **5年** | 期間満了日 | 労基法施行規則 |
| 健康診断個人票 | 5年 | 作成日 | 安衛法 |
| 扶養控除等申告書 | 7年 | 申告期限 | 国税通則法 |

## 2. デジタル化と原本廃棄のルール
- **電子帳簿保存法対応**:
  - スキャンデータは「真実性（タイムスタンプ）」と「可視性（検索性）」を確保すること。
  - 上記を満たす場合、紙の原本は即時廃棄可能。
- **廃棄手順**:
  - 保存期間満了したデータは、復元不可能な方法で削除し、削除ログを保存する。
  - 紙書類はシュレッダーまたは溶解処理を行い、廃棄証明書を取得する。

## 3. 特記事項 (2026年動向)
- 賃金請求権の時効（5年）への完全移行を視野に、全ての給与・労働時間関連書類は**「原則5年保存」**として運用することを強く推奨する。
