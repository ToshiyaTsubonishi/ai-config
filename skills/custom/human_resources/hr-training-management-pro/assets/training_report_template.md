# 研修アンケート・分析レポート テンプレート

## 1. 研修受講後アンケート (Forms設計案)
受講者の満足度だけでなく、行動変容を測定するための設問設計。

| カテゴリ | 設問内容 | 形式 |
| :--- | :--- | :--- |
| 満足度 | 今回の研修の内容は期待通りでしたか？ | 5段階評価 |
| 講師 | 講師の説明は分かりやすく、専門性が高かったですか？ | 5段階評価 |
| 理解度 | 研修の目的（[目的名]）を理解できましたか？ | 5段階評価 |
| **行動変容** | 研修で学んだ内容を、明日からの業務でどう活かしますか？ | **自由記述** |
| **障害要因** | 学んだ内容を実践する上で、障害となることは何ですか？ | 選択肢＋自由記述 |
| **提言** | 本研修をより良くするためのアドバイスをお願いします。 | 自由記述 |

## 2. 研修分析レポート (AI要約・提言構成)
AIがアンケート結果（CSV）を読み込んで生成するレポートの構成。

### 【エグゼクティブ・サマリー】
- 総合満足度スコア、理解度スコアの概況。
- AIによる自由記述のポジネガ分析（主要な不満点と評価点の要約）。

### 【分析詳細】
- 部署別・年次別の理解度差異の可視化。
- 行動変容の具体例（頻出キーワード：[例] 効率化、共有、マインドセット）。

### 【改善への提言】
- 次回開催に向けたコンテンツ修正案（例：演習時間の拡大、事例の最新化）。
- 現場での実践を阻む「障害要因」に対する組織的解決策の提案。
