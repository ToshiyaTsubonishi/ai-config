# デジタルスキル標準 (DSS) 2.0 準拠 スキルマッピング定義

## 1. 概要
経済産業省・IPA策定の「デジタルスキル標準 2.0」に基づき、人事部および全社員が身につけるべきDXリテラシーと推進スキルの評価軸を定義する。

## 2. DXリテラシー標準 (DSS-L) マッピング
全社員が習得すべき、DXの背景、データ活用、およびセキュリティ・倫理の項目。

| カテゴリ | 学習項目 (マインドセット・スキル) | 人事による評価基準 |
| :--- | :--- | :--- |
| **Why (背景)** | 社会におけるデータ活用の重要性の理解 | 外部環境変化に対するレポート作成能力 |
| **What (データ)** | データの種類、統計的な捉え方の基礎 | 表計算ソフトを用いた基本的データ分析 |
| **How (技術)** | AI、クラウド、IoT等の基本用語の理解 | 生成AIを業務に活用した事例の共有 |
| **Mindset** | 顧客中心、アジャイル、共創の姿勢 | PDCAサイクルの回数と改善提案数 |

## 3. DX推進スキル標準 (DSS-P) 役割定義
特にDXをリードする人材（DX推進担当、人事DX等）に向けた5つの役割。

1.  **ビジネスアーキテクト**: DXによる新ビジネスモデルや業務変革のグランドデザインを描く。
2.  **デザイナー**: ユーザー体験（UX）を重視し、従業員向けのシステムUI等を設計する。
3.  **データサイエンティスト**: 蓄積された人事データを解析し、経営に資するインサイトを抽出する。
4.  **ソフトウェアエンジニア**: Power AutomateやPAD等を用い、自律的に業務フローを実装する。
5.  **サイバーセキュリティ**: データ漏洩リスクを管理し、セキュアなDX環境を維持する。

## 4. 人事戦略への組み込み案
- **採用**: DSS-Pの役割に応じたキャリア採用の要件定義（JD）への反映。
- **育成**: 各役割に応じたeラーニング受講計画（LMS連携）。
- **評価**: DSSのリテラシー向上度を「専門性評価」の一部として採り入れる。
