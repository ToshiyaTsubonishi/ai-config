# 2025年1月 電子申請義務化 対応ガイドライン

## 1. 概要
デジタル手続法の施行に伴い、2025年1月1日より労働安全衛生法に基づく特定の報告書について、e-Gov等を通じた電子申請が全企業で原則義務化される。

## 2. 対象となる主な報告書
- **労働者死傷病報告** (休業4日以上)
- **定期健康診断結果報告書**
- **心理的な負担の程度を把握するための検査結果等報告書** (ストレスチェック)
- 総括安全衛生管理者・産業医等の選任報告

## 3. 人事担当者が行うべき準備
1. **gBizID / e-Govアカウントの確認**: 法人共通認証基盤の有効期限と権限設定。
2. **電子署名の準備**: 必要に応じて電子証明書の更新。
3. **入力支援サービスの活用**: 厚生労働省「帳票入力支援サービス」のブックマークと操作習熟。
4. **社内フローの刷新**: 被災発生から電子申請完了までのタイムライン（遅滞なく報告）の再定義。

## 4. 罰則・留意事項
- 義務化対象書類を書面で提出した場合、原則として受理されない（返送される）可能性がある。
- 虚偽の報告や報告の遅延は、労働安全衛生法違反（労働者死傷病報告の場合は50万円以下の罰金）の対象となる。
