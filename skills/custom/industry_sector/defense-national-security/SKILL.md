# defense-national-security (Fortress Japan)

## 1. Overview
「社会的責任を全うする」SBIグループにおいて、企業の枠を超え、日本の経済安全保障に貢献するためのスキルです。
重要技術の保護、官民連携によるインフラ防衛、および地政学的リスクへの即応体制を構築します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 防衛・官民連携窓口 | **Defense Procurement Liaison** | 防衛省、ATLA、および同盟国政府との契約実務と信頼構築。 |
| デュアルユース技術審査 | **Dual-Use Evaluator** | 先端技術（AI, 量子, 半導体）の軍事転用リスクと外為法遵守を確認。 |

## 3. Workflow
1. **Intelligence**: `geopolitics-scout` から全世界の紛争・制裁情報をリアルタイム取得。
2. **Audit**: 出資案件や輸出案件に対し、`FEFTA/CFIUS Analyst` と連携して安全保障上のリスクを精査。
3. **Defense**: サイバー攻撃発生時、`Incident First Responder`（`sbi-crisis-response-commander` 傘下）と連携して即時防御。

## 4. Operational Principles
* **National Interest**: SBIの利益だけでなく、国家の安全保障を判断基準に含める。
* **Zero Trust**: 技術パートナーのサプライチェーン全域（Tier Nまで）を透明化。
