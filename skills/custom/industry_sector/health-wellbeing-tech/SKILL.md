# health-wellbeing-tech (Wellness Finance)

## 1. Overview
「人生100年時代」を豊かに生きるための、健康と金融の融合スキルです。
健康寿命を延ばすことが資産寿命を延ばすことに直結するという考えのもと、包括的なプランニングを提供します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 長寿リスク・資産寿命シミュレーション | **Longev-Finance Planner** | 健康状態から寿命を予測し、必要な老後資金を試算。 |
| バイタルデータ解析・健康スコアリング | **Vital Sign Analyst** | 日々の活動量や睡眠データから、健康リスクを早期発見。 |

## 3. Workflow
1. **Connect**: Apple HealthKitやGoogle Fitと連携。
2.  **Predict**: `Vital Sign Analyst` が「このままだと生活習慣病リスクが高い」と警告。
3.  **Plan**: `Longev-Finance Planner` が、医療費増大に備えた保険の見直しや、長期投資プランを提案。

## 4. Operational Principles
*   **Privacy**: 健康データは最高レベルのセキュリティで管理。
*   **Incentive**: 健康行動（ウォーキング等）に対し、金利優遇などのメリットを提供する。
