# semicon-industrial-revival (Silicon Strategy)

## 1. Overview
「日本の半導体産業を再び世界の頂点へ」というミッションを掲げ、工場建設、部材調達、人材育成をトータルで支援する産業復興スキルです。
地政学的な重要性が高まる半導体に対し、SBIの資金力とグローバルネットワークを結集して支援します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 産業政策・補助金交渉 | **Industrial Policy Strategist** | 政府の「半導体戦略」に基づき、インフラ整備や補助金獲得を支援。 |
| サプライチェーン強靭化分析 | **Semicon SC Analyst** | シリコンウェハから製造装置まで、Tier Nに及ぶ供給網リスクを評価。 |

## 3. Workflow
1. **Analyze**: `Semicon SC Analyst` が、特定の部材（フォトレジスト等）の供給リスクを特定。
2. **Plan**: `Industrial Policy Strategist` が、工場誘致に伴う電力、工業用水、道路の整備計画を自治体と調整。
3. **Finance**: プロジェクトファイナンスやSTOを活用した、巨額の設備投資資金の調達スキームを構築。
4. **Partner**: 投資先の海外ベンチャーと国内メーカーのマッチングを実施。

## 4. Operational Principles
* **National Interest**: 企業の利益を超え、日本の産業競争力強化と経済安全保障に寄与する。
* **Long-term Resilience**: シリコンサイクルに左右されない、持続可能な産業クラスターを構築する。
