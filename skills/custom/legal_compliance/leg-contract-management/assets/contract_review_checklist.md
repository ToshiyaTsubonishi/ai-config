# 契約書レビュー・チェックリスト (Contract Review Checklist)

**契約名称:** ____________________
**相手方:** ____________________
**審査日:** 202_/__/__

## 1. 基本事項
- [ ] **当事者:** 甲乙の名称・住所は正しいか？（登記簿と一致）
- [ ] **目的:** 取引の内容が明確に定義されているか？
- [ ] **期間:** 契約期間と更新条件（自動更新の有無）は適切か？

## 2. 金銭・支払い
- [ ] **対価:** 金額、通貨、消費税の取り扱いは明記されているか？
- [ ] **支払条件:** 締め日、支払日、支払方法（振込手数料負担）は適切か？

## 3. リスク管理
- [ ] **損害賠償:** 上限規定（委託料の範囲内等）はあるか？ 無制限になっていないか？
- [ ] **契約解除:** 催告解除、無催告解除の事由は網羅されているか？
- [ ] **反社排除:** 暴力団排除条項（暴排条項）は含まれているか？
- [ ] **不可抗力:** 天災地変等の免責規定はあるか？

## 4. 知財・秘密保持
- [ ] **権利帰属:** 成果物の著作権はどちらに帰属するか？（受託なら自社帰属を目指す）
- [ ] **秘密保持:** 秘密情報の定義と管理義務は適切か？

## 5. 紛争解決
- [ ] **準拠法:** 日本法になっているか？
- [ ] **管轄:** 自社の本店所在地を管轄する裁判所（東京地裁等）になっているか？
