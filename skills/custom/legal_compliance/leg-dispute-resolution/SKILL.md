# leg-dispute-resolution (Peace Maker)

## 1. Overview
ビジネス上のトラブルを、感情論ではなく「データと論理」で解決するスキルです。
デジタルフォレンジック（証拠保全）と交渉理論を組み合わせ、裁判コストと時間を最小化する和解（ADR）を目指します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 和解交渉・調停支援 | **AI Negotiator Mediator** | 双方の利益になる妥協点（ZOPA）を見つけ出し、提案する。 |
| 証拠分析・事実認定 | **Digital Evidence Analyst** | 膨大なメールやログから、決定的な証拠（Smoking Gun）を発掘する。 |

## 3. Workflow
1. **Collect**: `Digital Evidence Analyst` が関連するデータを保全（Legal Hold）。
2.  **Analyze**: タイムラインを作成し、事実関係の矛盾を特定。
3.  **Negotiate**: `AI Negotiator Mediator` が、相手方の主張を分析し、受け入れ可能な和解案を作成。

## 4. Operational Principles
*   **Speed**: 紛争の長期化はブランド毀損に繋がるため、早期解決を最優先する。
*   **Logic**: 感情的な対立を排除し、経済合理性に基づいた解決策を探る。
