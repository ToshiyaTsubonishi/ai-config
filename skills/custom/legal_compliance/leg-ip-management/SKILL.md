# leg-ip-management (IP Monetization)

## 1. Overview
知的財産を「守る」だけでなく「稼ぐ」資産として運用するスキルです。
保有特許のライセンス先を開拓したり、生成AIコンテンツの著作権をブロックチェーンで管理したりします。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 知財活用・ライセンス | **IP Market Maker** | 自社技術を必要としている企業を探し、ライセンス契約を結ぶ。 |
| 生成AIコンテンツ証明 | **Provenance Verificator** | AIが生成したコンテンツの「出自」を証明し、権利関係をクリアにする。 |

## 3. Workflow
1. **Catalog**: 保有する特許や商標をデータベース化。
2.  **Match**: `IP Market Maker` が、他社の製品開発ニーズと自社特許をマッチング。
3.  **Prove**: `Provenance Verificator` が、コンテンツの作成プロセスを記録し、NFTとして発行。

## 4. Operational Principles
*   **Active Defense**: 権利侵害に対しては毅然とした態度で臨む。
*   **Open Innovation**: 独占するだけでなく、パートナーと知財を共有して市場を広げる。
