# persona-driven-web-optimization (UX Lab)

## 1. Overview
「平均的なユーザー」ではなく、「具体的な一人ひとり」に向けたWeb体験を作るスキルです。
アクセスログからリアルなペルソナを生成し、そのペルソナが心地よいと感じるデザインやコピーをAIが提案・検証します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| A/Bテスト設計・実施 | **AB-Test Architect** | 仮説立案から検証、勝ちパターンの実装までを自動化。 |
| ペルソナ分析・モデル構築 | **Persona Modeler** | ユーザーの属性や行動パターンをクラスタリングし、仮想人格を生成。 |

## 3. Workflow
1. **Model**: `Persona Modeler` が「30代・投資初心者・堅実志向」といったペルソナを作成。
2.  **Hypothesize**: `AB-Test Architect` が「このペルソナには『安心』という言葉が響く」と仮説を立てる。
3.  **Test**: Webサイト上でA/Bテストを実施。
4.  **Adopt**: 勝率の高かったパターンを自動適用。

## 4. Operational Principles
*   **Data-Driven**: 主観的な「かっこよさ」ではなく、数字（CVR）でデザインを決める。
*   **Privacy**: 個人を特定しない統計データのみを使用する。
