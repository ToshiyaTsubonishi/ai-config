# サステナビリティ・ESG戦略 高度化マニュアル

## 1. 高度なGHG排出量算定 (Measurement)
- [ ] **Scope 3 Category 15**: PCAF（金融のための炭素会計パートナーシップ）基準に基づき、投融資先の排出量をアセットクラス（上場株式、ビジネスローン等）ごとに算定。
- [ ] **Data Quality Score**: 投融資先の公開データ、推計値、およびエンゲージメントによる実測値の比率を管理し、スコアを向上。
- [ ] **Carbon Footprint**: 商品・サービスごとの「炭素の見える化」と、顧客へのフィードバック機能の検討。

## 2. インターナルカーボンプライシング (ICP) 運用基準
- **価格設定**: 1トンあたり1万円（または国際トレンドに合わせた動的設定）。
- **投融資審査への統合**:
    - 炭素調整後ROICの算出。
    - 高排出セクターへの新規融資に対する「炭素プレミアム（追加コスト）」の仮想計上。

## 3. 気候・自然シナリオ分析 (Scenario Analysis)
- **Transition Risk**: 炭素税導入および技術革新に伴う、投融資先の信用格付けへの影響評価。
- **Physical Risk**: 大規模洪水・高潮による担保不動産の毀損確率（ハザードマップ連携）と予想損失額(EL)の算定。
- **TNFD Integration**: 自然資本（水資源、生物多様性）の依存度・影響度のLEAPアプローチによる評価。

## 4. 戦略的開示 & 移行計画 (Disclosure)
- [ ] **Transition Plan**: 2050年ネットゼロに向けた「セクター別削減マイルストーン」の策定。
- [ ] **Greenwashing Check**: 開示内容と実際の投融資行動の乖離をコンプライアンス部門が定期レビュー。
- [ ] **Sustainable Finance**: グリーンボンド、サステナビリティ・リンク・ローンの組成目標達成状況の公開。
