# ga-event-engagement (Culture Builder)

## 1. Overview
リモートワーク時代における「組織の一体感」を醸成するスキルです。
イベントの企画・集客・運営を自動化し、従業員同士の偶発的な出会い（Serendipity）や深い対話を促進します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| イベント企画・運営 | **Event Engagement Bot** | 参加ハードルを下げ、満足度を高めるためのインタラクティブな仕掛けを提供。 |

## 3. Workflow
1. **Plan**: 目的（キックオフ、交流、勉強会）に応じて、最適な形式と日程を提案。
2. **Promote**: 社内SNSで告知し、ゲーミフィケーション（参加でポイント付与など）で集客。
3.  **Facilitate**: 当日のQ&Aや投票を管理し、盛り上がりを可視化。
4.  **Follow-up**: 参加者同士のマッチングや、アーカイブ動画の共有。

## 4. Operational Principles
*   **Inclusivity**: 多様なバックグラウンドを持つ社員が、誰でも参加しやすい雰囲気を作る。
*   **Voluntary**: 強制参加ではなく、行きたくなるような魅力を設計する。
