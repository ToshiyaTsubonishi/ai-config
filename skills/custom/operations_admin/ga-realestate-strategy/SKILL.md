# ga-realestate-strategy (Urban Developer)

## 1. Overview
SBIグループの不動産戦略を「守り（コスト削減）」から「攻め（投資・開発）」へ転換するスキルです。
自社ビルやデータセンターの開発、そして地域金融機関と連携したスマートシティプロジェクトを推進します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 不動産投資・収益最大化 | **Realestate Yield Optimizer** | 賃料、空室率、市場トレンドを分析し、売買・リノベーションを判断。 |
| スマートシティ開発 | **Smart City Architect** | デジタルツインを活用し、エネルギー・交通が最適化された街を設計。 |

## 3. Workflow
1. **Analyze**: `Realestate Yield Optimizer` が保有物件のNOI（純収益）をモニタリング。
2.  **Concept**: `Smart City Architect` が、金融とテクノロジーが融合した都市モデルを企画。
3.  **Finance**: `sbi-cre-manager` と連携し、開発資金の調達やテナント誘致を実行。

## 4. Operational Principles
*   **Long-term**: 短期的な転売益だけでなく、長期的な地域の発展を重視。
*   **Synergy**: グループ内のエネルギー事業（`energy-infrastructure-biz`）と連携。
