# 総務・管財 & ガバナンス 運用マニュアル

## 1. 株主総会 電子提供制度対応チェックリスト
- [ ] **適時掲載 (T-21)**: 招集通知発送日のいずれか早い日までに、自社HPおよび東証閲覧サービスへ掲載完了。
- [ ] **書面交付対応**: 書面交付請求者へのフルセット冊子の発送管理（期限厳守）。
- [ ] **可用性確保**: サーバーダウン時に備えた代替サイト（ミラーサイト）の稼働確認。
- [ ] **ESG Q&A**: 人的資本（女性管理職比率、男性育休等）および気候変動対応（TCFD）の回答準備。

## 2. インテリジェント・ファシリティ管理 (FM)
- **ワークプレイス最適化**: 出社率データの分析に基づくオフィス面積の適正化およびフリーアドレス制の運用。
- **環境配慮 (Green Office)**: 逵√お繝肴ｳ募ｯｾ蠢懊髮ｻ蜉帑ｽｿ逕ｨ驥上Δ繝九ち繝ｪ繝ｳ繧ｰと、LED化等の省エネ施策。
- **資産棚卸**: 固定資産（什器・PC等）のRFID/バーコードを用いた効率的な現物照合プロセス。

## 3. 文書・印章・事務管理
- [ ] **電子保存**: タイムスタンプ付与による「電子帳簿保存法」スキャナ保存要件の遵守。
- [ ] **押印申請フロー**: 決裁権限規定に基づく電子承認および物理印章の施錠管理。
- [ ] **車両管理**: 運行日誌のデジタル化およびアルコールチェック記録のクラウド保存。

## 4. BCP初動アクションプラン
1. **Safety Confirmation**: 安否確認システムの自動発動および全役職員の回答状況把握。
2. **Command Center**: 危機対策本部の設置（物理/仮想）および各拠点との通信確保。
3. **Critical Process Recovery**: 優先復旧業務（決済、窓口対応等）のRTO（目標時間）内の再開。
