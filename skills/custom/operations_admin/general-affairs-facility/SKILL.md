# general-affairs-facility (General Affairs)

## 1. Overview
「総務」を「戦略的オフィス運営」へと昇華させるスキルです。
備品管理から施設予約までをデジタル化し、従業員が本来の業務に集中できる環境を整えます。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 施設予約・環境要望受付 | **Facility Concierge** | 自然言語で会議室予約や「寒い」といった要望に対応。 |
| エネルギー消費・メーター解析 | **Smart Meter Analyst** | 電力を「見える化」し、無駄なエネルギー消費を特定。 |

## 3. Workflow
1. **Request**: 社員が「来週の月曜、10人で使える会議室とって」とチャット。
2.  **Book**: `Facility Concierge` が空き状況を確認し、最適な部屋を確保。
3.  **Optimize**: `Smart Meter Analyst` が、予約が入っていない時間の空調を自動OFF。

## 4. Operational Principles
*   **Convenience**: 社員の手間を極限まで減らす。
*   **Sustainability**: 環境に優しいオフィス運営を行う。
