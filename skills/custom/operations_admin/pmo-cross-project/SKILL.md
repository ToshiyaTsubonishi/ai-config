# pmo-cross-project (Program Management)

## 1. Overview
「木を見て森も見る」視点で、組織全体のプロジェクトポートフォリオを最適化するスキルです。
特定のプロジェクトの遅延が他にどう波及するかを予測し、リソース（人・金）の再配分をダイナミックに行います。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| リソース配分・外注管理 | **Hybrid Workforce Orchestrator** | 社員、AI、外部パートナーの最適な組み合わせを提案。 |
| プロジェクト間依存関係管理 | **Inter-Project Dependency Tracker** | クリティカルパスの連鎖を可視化し、ドミノ倒しを防ぐ。 |

## 3. Workflow
1. **Map**: `Inter-Project Dependency Tracker` が全プロジェクトのマイルストーンをリンク。
2.  **Detect**: 「プロジェクトAの遅れにより、プロジェクトBの開始が遅れる」リスクを検知。
3.  **Resolve**: `Hybrid Workforce Orchestrator` が、余裕のあるチームから人員を融通するか、外部リソースを活用するかを判断。

## 4. Operational Principles
*   **Prioritization**: 戦略的需要度の高いプロジェクトにリソースを集中させる。
*   **Visibility**: 隠れた依存関係（暗黙の前提）をなくす。
