# banking-regional-dx (Regional Revitalization)

## 1. Overview
地方創生への貢献をミッションとし、地域金融機関のデジタルシフトを加速させるスキルです。
SBIグループの成功モデルを横展開し、地銀を「地域の情報・決済ハブ」へと進化させます。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| BaaS導入・クラウド移行 | **BaaS Architect** | 銀行機能をモジュール化し、非金融事業者への埋め込みを支援。 |
| スーパーアプリ・顧客接点強化 | **Super App Producer** | 住民の生活に密着したアプリ（地域生活OS）の企画・運営。 |
| 法人向けデータ営業支援 | **Regional Sales Strategist** | 商流データを解析し、ビジネスマッチングや融資案件を自律化。 |

## 3. Workflow
1. **Gap Analysis**: dx-maturity-auditor と連携し、対象銀行の課題を特定。
2. **Implementation**: 最適なエージェントを配備し、BaaS基盤からアプリまでを一気通貫で構築。
3. **Evolution**: egional-bank-self-evolution 連携により、自律的なサービス改善ループを確立。

## 4. Operational Principles
* **Partnership**: 地方銀行の独立性を尊重しつつ、シナジーを最大化。
* **Customer Centricity**: 地域住民の利便性を最優先した設計。
