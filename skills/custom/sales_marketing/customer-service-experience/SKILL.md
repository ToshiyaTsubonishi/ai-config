# customer-service-experience (Empathy AI)

## 1. Overview
「顧客中心主義」を体現する、AI時代のカスタマー・エクスペリエンス（CX）基盤です。
単なる自動応答ではなく、顧客の「不」を解消し、感動を呼ぶ「最高の一手（Next Best Action）」を提示します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 感情認識・コンシェルジュ対応 | **Emotion-aware Concierge** | 声のトーンや言葉遣いから心理状態を読み取り、対応を最適化。 |
| エスカレーション最適化 | **Escalation Orchestrator** | 複雑な案件や感情的な課題を、コンテキストを維持したまま有能な人間へ。 |

## 3. Workflow
1. **Listen**: `vital-sign-analyst` 連携により（許諾を得た範囲で）ユーザーの焦りなどを検知。
2. **Retrieve**: `sbi-encyclopedia-builder` から最適なナレッジを検索。
3. **Act**: `Persona Adapter`（`tech-natural-language-ops` 傘下）を用いて、顧客に最も響く話し方で回答。

## 4. Operational Principles
* **Empathy Over Speed**: 速さよりも、納得感と満足度を重視。
* **Invisible Finance**: 顧客が気づかないうちに問題が解決されている「究極の利便性」を目指す。
