# sales-enterprise-strategy (Account Based Marketing)

## 1. Overview
大企業特有の「複雑な意思決定プロセス」を解き明かし、組織ぐるみで攻略するスキルです。
キーマンの特定から、部門間の政治力学を考慮した提案シナリオの作成まで、戦略的な営業活動を支援します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 意思決定者分析・攻略 | **DMU Navigator Bot** | 組織図を解析し、誰が決定権を持っているか（Champion/Economic Buyer）を特定。 |
| 戦略的提携・大型案件発掘 | **Strategic Deal Hunter** | 顧客の中期経営計画を読み解き、アライアンスやJVの提案機会を探る。 |

## 3. Workflow
1. **Map**: `DMU Navigator Bot` が、顧客の組織図上に「賛成派」「反対派」をマッピング。
2.  **Plan**: `Strategic Deal Hunter` が、顧客の経営課題（コスト削減、新規事業）に合致するSBIのアセットを選定。
3.  **Approach**: 役員ルートや既存取引を活用し、トップ面談を設定。

## 4. Operational Principles
*   **Long-term**: 目先の売上よりも、顧客との長期的なパートナーシップ（LTV）を重視する。
*   **Team Selling**: 営業マン個人ではなく、チームで情報を共有して戦う。
