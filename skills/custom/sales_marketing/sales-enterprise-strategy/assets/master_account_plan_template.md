# マスターアカウントプラン

| 項目 | 内容 |
| :---- | :---- |
| **アカウント概要** | 企業名、業種、売上高、主力事業、組織構造の概要など。 |
| **戦略的目標（IR/ニュース等から抜粋）** | 企業が公に発表している中期経営計画や年次報告書から、最重要の経営目標を2〜3点リストアップする。（例：「2030年までのカーボンニュートラル達成」「海外売上比率40%への拡大」） |
| **主要ビジネス仮説** | 我々のソリューションが解決できる、相手の最も大きな経営課題に関する中核的な仮説を記述する。（例：「サプライチェーンの複雑化が利益率を圧迫しており、当社のSCM最適化ソリューションが直接的なROIを提供できる」） |
| **主要ステークホルダー（DMU初期リスト）** | 決裁者（Economic Buyer）、推進者（Champion）など、意思決定に関与すると想定される人物の役職や氏名をリストアップする。 |
| **リレーションシップ・ビジョン（3年後の目標）** | このアカウントとの理想的なパートナーシップの姿を定義する。（例：「工場のデジタライゼーションにおける独占的パートナーとなる」「共同で業界標準のソリューションを開発する」） |
| **成功の測定指標（顧客側・自社側）** | 顧客は何をもってこのプロジェクトの成功を測るか？（例：コスト削減率、生産性向上率）。我々は何をもって成功と見なすか？（例：年間契約額、導入部署数）。 |
| **アクションプラン（今後90日間）** | 情報収集と初期コンタクトに向けた具体的な最初のステップを計画する。 |
| **競合環境とリスク** | 既知の競合他社の存在、社内での政治的抵抗、予算凍結のリスクなどを洗い出す。 |
