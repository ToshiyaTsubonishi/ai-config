# Playbook: Regional Bank Penetration Strategy

**Source:** Based on Project Gateway (Shikoku, 2026)
**Objective:** 排他的な地銀連合（Alliance）を攻略し、SBI経済圏へ取り込む。

---

## Phase 1: The Wedge (一点突破)
*   **Target**: エリア内の **Tier 1 (第一地銀)** または **Innovator (改革派)**。
*   **Tactic**: 銀行単独では絶対に開発できない「飛び道具」を提供する。
    *   *Examples*: Web3ウォレット, 大阪万博NFT, 半導体サプライチェーン金融。
*   **Goal**: 「あの銀行がSBIと組んで新しいことを始めた」というニュースを作る。

## Phase 2: The Hub (ハブの活用)
*   **Target**: 既にSBIと提携している、または親和性の高い **Hub Bank**。
*   **Tactic**: **Referral (紹介)** の仕組みを制度化する。
    *   *Incentive*: 紹介元の銀行に対し、共同ファンドの販売枠や、先進事例としてのPR機会を提供する。
*   **Goal**: 「SBIの営業」としてではなく、「仲間の地銀からの紹介」としてターゲット行にアプローチする。

## Phase 3: The Encirclement (包囲網)
*   **Target**: 保守的な **Laggards (追随層)**。
*   **Tactic**: **Peer Pressure (同調圧力)** を利用する。
    *   *Event*: エリア全行の役員を集めた「サミット」を開催し、先行銀行（Phase 1）に登壇させる。
    *   *Logic*: 「他行はもうやっています。御行だけ乗り遅れても良いのですか？」
*   **Goal**: リスク回避志向の銀行を、恐怖心（FOMO）を利用してクロージングする。

---
## Checklist for Each Phase

### Phase 1
- [ ] 地域のTier 1銀行はどこか？
- [ ] その銀行のDX担当役員は誰か？
- [ ] 「国家プロジェクト級」のネタ（万博、半導体）はあるか？

### Phase 2
- [ ] 既存提携行（または出資行）はエリア内にあるか？
- [ ] その担当者とホットラインはあるか？
- [ ] 紹介することのメリット（インセンティブ）は明確か？

### Phase 3
- [ ] サミット（イベント）の開催場所・日時は？
- [ ] 登壇者（先行銀行役員）の承諾は取れたか？
- [ ] クロージング用の「即決オファー（無償トライアル等）」は準備できたか？
