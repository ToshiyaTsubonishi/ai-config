# Validation Checklist for SALES-Enterprise-Strategy

## 1. 構造確認
- [ ] `SKILL.md` が存在し、6名のAIエージェントの役割が定義されているか。
- [ ] `assets/` フォルダに6つのテンプレートが存在するか。
- [ ] `references/` フォルダにマニュアルが存在するか。

## 2. プロセス実行テスト
- [ ] **Phase 0:** `master_account_plan_template.md` を用いて、架空または実際の企業のターゲット選定ができるか。
- [ ] **Phase 1:** インテリジェンスアナリストAIが中計から仮説を導き出せるか。
- [ ] **Phase 2:** フィールドセールスAIがDMUマップを埋められるか。
- [ ] **Phase 3:** バリューコンサルタントAIがキャンバスを作成し、ナラティブを生成できるか。
- [ ] **Phase 4:** MEDDICチェックリストを用いて、案件の欠落情報を特定できるか。
- [ ] **Phase 5:** 想定問答集を作成し、シミュレーションが行えるか。

## 3. ログ保存テスト
- [ ] 議論の過程が `log/` ディレクトリに保存されるか。
- [ ] 最終成果物が `output/` ディレクトリに保存されるか。
