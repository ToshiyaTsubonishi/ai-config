# sales-marketing-ops (Revenue Machine)

## 1. Overview
「マーケティング」と「セールス」の壁を壊し、売上（Revenue）を科学的に最大化するスキルです。
見込み客（リード）の獲得、育成、そして商談化までのプロセスを数値で管理し、ボトルネックを解消します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| コンテンツ制作・配信 | **Content Engine Bot** | 顧客の関心を惹くホワイトペーパーや記事を量産。 |
| リード育成・スコアリング | **Lead Nurturer Agent** | メールの開封やWeb閲覧履歴から「今すぐ客」を見極める。 |
| 競合分析・営業戦略 | **Sales Intelligence Expert** | 競合の動きを分析し、勝率の高いトークスクリプトを作成。 |

## 3. Workflow
1. **Attract**: `Content Engine Bot` がSEO記事を作成し、リードを獲得。
2.  **Nurture**: `Lead Nurturer Agent` が、顧客の興味に合わせたステップメールを配信。
3.  **Qualify**: スコアが高まったリードを、インサイドセールスに通知。
4.  **Close**: `Sales Intelligence Expert` が提供する競合比較資料を使ってクロージング。

## 4. Operational Principles
*   **Data Integrity**: MAツールとSFAのデータ連携を密にし、情報の分断を防ぐ。
*   **Speed**: ホットなリードには、5分以内にアプローチする体制を作る。
