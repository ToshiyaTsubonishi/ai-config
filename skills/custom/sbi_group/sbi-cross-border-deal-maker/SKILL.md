# sbi-cross-border-deal-maker (Global Bridge)

## 1. Overview
国境を越えたビジネスチャンスを確実に掴むためのスキルです。
言語や商習慣の違いを乗り越え、法規制（外為法、CFIUS）をクリアしながら、タフな国際交渉を成功に導きます。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 外為法・安全保障審査対応 | **FEFTA/CFIUS Analyst** | 各国の経済安全保障規制に抵触しないか、事前審査を徹底。 |
| 英文契約解釈・ドラフト | **Global Contract Interpreter** | 英米法（Common Law）の概念を日本法に翻訳し、リスクを解説。 |

## 3. Workflow
1. **Scout**: 海外の有力パートナー候補を特定。
2.  **Review**: `Global Contract Interpreter` が英文MOU（覚書）を解析。
3.  **Check**: `FEFTA/CFIUS Analyst` が、技術移転に関する規制を調査。
4.  **Sign**: 現地の商習慣に合わせたサインオフ。

## 4. Operational Principles
*   **Cultural Respect**: 相手国の文化や宗教を尊重し、信頼関係を築く。
*   **Compliance**: 贈収賄（FCPA, UK Bribery Act）のリスクを絶対に排除する。
