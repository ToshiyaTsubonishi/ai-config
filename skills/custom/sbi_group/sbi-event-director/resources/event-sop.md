# Event SOP (Standard Operating Procedure)

**Purpose**: Quality control for internal and external events.
**Consolidated from**: Tasks 11, 12, 19, 20.

## 1. VIP "Omotenashi" Protocol
*   **Arrival**: Use "Security Elevator" (Reserve mode).
*   ** refreshments**: 
    *   Standard: Coffee/Tea.
    *   VIP: Matcha + Higashi (Toraya).
*   **Souvenir**: SBI Novelty (High-end Pen or Imabari Towel).

## 2. Internal Party Guidelines
*   **Budget**: Max 5,000 JPY/person (Welfare expense limit).
*   **Venue**: Prioritize Internal Cafeteria (Cost 0).
*   **Alcohol**: Allowed after 18:30. "Moderate" amount only.

## 3. Press Conference Checklist
*   [ ] **Press Release**: Distributed via PR TIMES.
*   [ ] **Reception**: Media List check (Business Card required).
*   [ ] **Wi-Fi**: Dedicated SSID "SBI_PRESS" (5GHz, no cap).
*   [ ] **Security**: Bag check for cameras/recording gear (No Spy Cams).

## 4. Safety & Insurance
*   **Sports Events**: Sports Safety Insurance (Spot coverage) is MANDATORY.
*   **First Aid**: Nurse station or AED availability check (`task_28`).
