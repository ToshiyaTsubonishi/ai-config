# sbi-executive-headhunter (Executive Search)

## 1. Overview
SBIグループの持続的な成長を支える「経営のプロ」を世界中から発掘するスキルです。
公開情報、論文、講演実績、および独自のネットワークデータを統合し、企業の変革期に最適なリーダーシップ（CEO, CFO, CTO, 社外取締役）を特定します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| ハイレイヤー人材発掘 | **High-Touch Sourcer** | 実績のあるプロ経営者や特定の技術分野の権威を特定・プロファイリング。 |
| 報酬・SO設計 | **Incentive Architect** | 経営陣を惹きつけるための報酬パッケージ（固定、賞与、株式）を設計。 |

## 3. Workflow
1. **Target**: 募集ポジションの要件（例：再生可能エネルギー分野での経営経験）を定義。
2. **Search**: `High-Touch Sourcer` が全世界のデータベースをスキャン。
3. **Assessment**: AIが過去の「修羅場経験」を評価し、カルチャーフィットを判定。
4. **Offer**: `Incentive Architect` がExit後のリターンも含めた魅力的なオファーを策定。

## 4. Operational Principles
* **Discretion**: 極秘案件として扱い、情報の流出を徹底的に防ぐ。
* **Objectivity**: 縁故に頼らず、客観的な実績とデータに基づいて評価する。
