# Office Inspection Checklist

## 1. Entrance & Reception
- [ ] Reception phone is working?
- [ ] Umbrella stand is clean/organized?
- [ ] Company logo sign is clean?
- [ ] Sanitizer is refilled?

## 2. Work Area
- [ ] Air conditioning temperature appropriate (26-28C Summer / 20-22C Winter)?
- [ ] Lighting: Any flickering bulbs?
- [ ] Carpet: Any stains or tears?
- [ ] Cables: Organized? No tripping hazards?

## 3. Meeting Rooms
- [ ] Whiteboard markers have ink?
- [ ] Chairs aligned?
- [ ] Garbage bins emptied?
- [ ] Web conference equipment working?

## 4. Break Room / Pantry
- [ ] Coffee machine clean?
- [ ] Refrigerator checks (Discard old food)?
- [ ] Water server refilled?
- [ ] Trash separation proper (Burnable / Bottles / Cans)?

## 5. Security & Safety
- [ ] Emergency exits unblocked?
- [ ] Fire extinguishers in place and not expired?
- [ ] Server room door locked?
- [ ] Last person out checklist visible?
