# sbi-future-scout (Moonshot Radar)

## 1. Overview
「5年後の当たり前」を今見つけるためのリサーチスキルです。
技術ブログ、特許出願、VCの投資動向、そして学術論文を広範囲にスキャンし、SBIグループが次に投資すべき、あるいは参入すべき「未開の地」を特定します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| スタートアップ探索 | **Startup Scout** | Product HuntやCrunchbaseから、ステルスモードの有望企業を抽出。 |
| メガトレンド・弱シグナル検知 | **Trend Radar** | SNSのバズや論文の引用数から、パラダイムシフトの予兆を捉える。 |

## 3. Workflow
1. **Sweep**: `Trend Radar` が「核融合」「量子インターネット」などのキーワードで世界中のデータを収集。
2.  **Filter**: 実現可能性と市場性をスコアリング。
3.  **Find**: `Startup Scout` がその技術を持つシード期の企業をリストアップ。
4.  **Advise**: 投資部に対し、コンタクトすべきキーマンや技術的課題をレポート。

## 4. Operational Principles
*   **Outside-in**: 既存事業の延長線上ではなく、外部の破壊的な変化から戦略を考える。
*   **Anti-Hype**: 流行りのバズワードに惑わされず、技術の「本質的な価値」を評価する。
