# sbi-group-unified-strategy (Unified Proposal)

## 1. Overview
大企業（エンタープライズ）や自治体に対し、「SBIグループ」を一つの巨大なソリューションとして提案するスキルです。
個社の営業ではなく、グループ全体のトップセールスを支援し、金融から先端技術、インフラまでを網羅した「唯一無二」の提案を構築します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 顧客分析・経営課題深掘 | **Customer Insight Miner** | 顧客の中計や有報から、経営層が真に解決したい「痛み」を特定。 |
| 統合提案・スキーム設計 | **Unified Solution Architect** | 銀行、証券、保険、Web3等のアセットを最適に組み合わせた提案書を作成。 |

## 3. Workflow
1. **Analyze**: `Customer Insight Miner` が顧客の最新動向をリサーチ。
2.  **Ideate**: 関連するグループ各社の担当エージェントを招集し、ブレスト。
3.  **Design**: `Unified Solution Architect` が、複数の契約やプロダクトを一本化した統合提案を構築。
4.  **Present**: トップ会談用のエクイティ・ストーリー（`equity-story-teller` 連携）を作成。

## 4. Operational Principles
*   **One SBI**: 顧客から見て窓口を一元化し、利便性を高める。
*   **Social Impact**: 企業の利益だけでなく、その提携が社会にどのようなインパクトを与えるかを語る。
