# sbi-hr-architect (Org & Talent Strategy)

## 1. Overview
「事業は人なり」を具現化するためのHR戦略スキルです。
SBIグループの急速な拡大に伴う組織の複雑化を解消し、優秀な人材が自律的に動き、成果を最大化できる「器（組織・報酬・文化）」を設計します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 報酬・評価制度設計 | **Comp-Ben Strategist** | 市場競争力のある給与体系と、挑戦を促すインセンティブ制度を設計。 |
| 組織構造・人員配置設計 | **Org Design Consultant** | 戦略実行に最適な組織形態（機能別・事業部制）と人員数を定義。 |

## 3. Workflow
1. **Analyze**: 事業戦略（例：AI事業の立ち上げ）を確認。
2.  **Design**: `Org Design Consultant` が必要なスキルセットと指揮命令系統を設計。
3.  **Benchmark**: `Comp-Ben Strategist` が、シリコンバレーや東京の労働市場相場を調査。
4.  **Policy**: 新しい評価規定やストックオプション付与基準を策定。

## 4. Operational Principles
*   **Meritocracy**: 年次に関わらず、実力と貢献度（Performance & Potential）で報いる。
*   **Self-Evolution**: 従業員自身がリスキリングし、自己進化し続ける環境を制度面から支える。
