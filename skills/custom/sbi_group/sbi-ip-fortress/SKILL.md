# sbi-ip-fortress (IP Shield)

## 1. Overview
「攻め」の知財戦略だけでなく、「守り」の知財戦略を自動化するスキルです。
自社の技術領域をカバーする特許網（パテント・ポートフォリオ）を構築し、模倣品の出現やブランド毀損を24時間監視します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| パテントマップ・網羅的分析 | **Patent Landscape Analyst** | 競合の出願を監視し、自社の技術的な立ち位置を明確化。 |
| ブランド侵害監視 | **Trademark Brand Monitor** | 世界中のECサイトやドメインを巡回し、偽ブランド品を検知。 |

## 3. Workflow
1. **Monitor**: `Trademark Brand Monitor` が「SBI」ロゴを無断使用しているサイトを発見。
2.  **Analyze**: `Patent Landscape Analyst` が、競合他社が自社の特許周辺に出願してきたことを検知。
3.  **Action**: 侵害警告書の送付や、異議申し立ての手続きをドラフト作成。

## 4. Operational Principles
*   **Deterrence**: 「権利侵害をしたら即座に見つかる」と思わせる抑止力を持つ。
*   **Cost-Benefit**: 訴訟コストと被害額を比較し、経済合理性のある対応を行う。
