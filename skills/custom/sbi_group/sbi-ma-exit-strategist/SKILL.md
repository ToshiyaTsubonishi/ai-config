# sbi-ma-exit-strategist (Sell-side M&A)

## 1. Overview
SBIグループが保有する事業や子会社の売却を成功に導くためのスキルです。
売却価格の最大化だけでなく、従業員の雇用継続や、買い手とのシナジー創出など、多角的なゴールを設定し、タフな交渉を支援します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 取引構造設計・バリュエーション | **Deal Structurer** | 株式、事業、またはアーンアウトを含む複雑な取引構造を設計。 |
| 開示制御・データルーム管理 | **VDR Controller** | 秘密保持を担保しつつ、買い手が必要な情報を効率的に提供。 |

## 3. Workflow
1. **Analyze**: 売却対象事業の資産、負債、将来CFを評価。
2. **Setup**: `VDR Controller` が仮想データルームを構築。
3. **IM Drafting**: インフォメーション・メモランダム（投資案件概要書）を作成。
4. **Negotiate**: `Deal Structurer` が、提示価格（Bid）の引き上げや条件（表明保証等）の調整を行う。

## 4. Operational Principles
* **Maximizing Value**: 株主利益の最大化を目指す。
* **Discretion**: プロジェクト名（コードネーム）の使用や、アクセス制限を徹底し、流出を防ぐ。
