# sbi-post-merger-integration (Value-up)

## 1. Overview
M&Aの成功を確定させるための「統合」スキルです。
Day 1（買収初日）からの100日間で、SBIグループの管理体制（ガバナンス）を導入し、当初計画していたシナジー（相乗効果）を具現化するためのハンズオン支援を行います。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| ガバナンス・管理体制導入 | **Governance Setup Agent** | 稟議、会計、コンプライアンスの仕組みをSBI標準へ迅速に移行。 |
| シナジー創出・実行支援 | **Synergy Realizer** | 顧客基盤の共有、共同製品開発など、売上・コスト両面のシナジーを実現。 |

## 3. Workflow
1. **Prepare**: 買収完了前に、新旧組織の差異を分析。
2. **Onboard**: `Governance Setup Agent` が管理規定を配布し、説明会を実施。
3. **Connect**: `Synergy Realizer` が、SBIグループ各社とのビジネスマッチングをコーディネート。
4.  **Track**: 統合効果（シナジー達成度）をKPIとして毎月報告。

## 4. Operational Principles
* **People First**: 買収先社員の不安を取り除き、モチベーションを高めるコミュニケーションを重視。
* **Fast Implementation**: 仕組みの統合は「速さ」が最重要。時間をかけるほど価値が減少する。
