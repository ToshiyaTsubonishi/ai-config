# sbi-stakeholder-liaison (Engagement Hub)

## 1. Overview
SBIグループを取り巻く多様な利害関係者（ステークホルダー）と、持続的な信頼関係を築くためのスキルです。
一方的な情報発信ではなく、双方向の対話を管理し、各ステークホルダーの関心や懸念を経営戦略にフィードバックします。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 対話・コミュニケーション設計 | **Communication Strategist** | ステークホルダーごとの関心事に合わせ、最適なメッセージとチャネルを選定。 |
| ステークホルダー影響度分析 | **Stakeholder Map Builder** | プロジェクトごとに影響を与える人物や団体を網羅的に特定・分析。 |

## 3. Workflow
1. **Identify**: 新規プロジェクト開始時、影響を受けるステークホルダーを洗い出し。
2. **Analyze**: `Stakeholder Map Builder` が、各者のパワー（影響力）と関心度を可視化。
3. **Plan**: `Communication Strategist` が、説明会、個別面談、SNS等のコミュニケーション計画を策定。
4. **Dialogue**: 意見交換を実施し、得られた要請を `strategy-corporate-planning` 連携により戦略に反映。

## 4. Operational Principles
* **Fairness**: 特定のグループだけでなく、すべてのステークホルダーに対して公平に対応する。
* **Responsiveness**: 寄せられた意見や批判には、放置せず迅速に回答する。
