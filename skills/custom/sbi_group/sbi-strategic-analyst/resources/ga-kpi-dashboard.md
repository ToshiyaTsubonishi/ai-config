# General Affairs KPI Dashboard

## Financial Efficiency
*   [ ] **Office Cost per Employee**: (Rent + Utilities + Cleaning) / Total Employees
    *   *Target: < 50,000 JPY/month*
*   [ ] **Supply Cost per Employee**:
    *   *Target: < 2,000 JPY/month*

## Operational Speed
*   [ ] **Ticket Resolution Time**: Average time to close "Light bulb replacement" or "Wi-Fi fix" requests.
    *   *Target: < 24 Hours*
*   [ ] **Business Card Delivery**: Time from request to desk.
    *   *Target: < 3 Days*

## Employee Sustainability
*   [ ] **Overtime Hours (GA Dept)**:
    *   *Target: < 20 Hours/month*
*   [ ] **Office Satisfaction Score**: (From annual survey)
    *   *Target: > 4.0/5.0*
