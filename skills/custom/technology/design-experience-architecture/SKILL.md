# design-experience-architecture (Spatial & NoUI)

## 1. Overview
「金融を透明にする（Invisible Finance）」ことを目指し、画面の中（UI）から生活空間（Spatial）全体へと体験を広げるスキルです。
Apple Vision Pro等の空間コンピューティングや、音声・直感による操作（NoUI）を設計します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| 非接触・音声・意図予測UI | **NoUI Strategist** | ユーザーが「操作」を意識せずとも望む結果が得られるUXの追求。 |
| 空間UX・メタバース店舗設計 | **Spatial UX Designer** | 3D空間における資産の可視化と、アバターを通じた高度な対話体験。 |

## 3. Workflow
1. **Journey Mapping**: 顧客の「24時間」をモデル化し、金融が必要になる「瞬間（Micro-moments）」を特定。
2. **Prototype**: `Creative Asset Generator` 連携により、メタバースやMR空間のプロトタイプを高速生成。
3. **Refine**: `Silver UI Designer`（`tech-frontend-engineering-accessible` 傘下）と協力し、全世代に使いやすい直感性を確保。

## 4. Operational Principles
* **Atmospheric Design**: 金融サービスが空気のように環境に溶け込んでいる状態。
* **Multi-sensory**: 視覚だけでなく、音や触覚（Haptics）を活用した信頼の構築。
