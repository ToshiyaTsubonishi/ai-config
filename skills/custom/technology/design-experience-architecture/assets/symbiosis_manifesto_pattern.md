# Symbiosis Manifesto Page Pattern

## 1. Purpose
In the AGI era, a company's "Mission Page" must be replaced by a "Manifesto Page".
It does not describe *what we do*, but *the philosophy of our existence* alongside AGI.

## 2. Structure Pattern (The Golden Flow)

### Section 1: The Declaration (Hero)
*   **Visual:** Abstract, cinematic background (Void/Space).
*   **Copy:** A bold statement of "Symbiosis" or "Singularity".
*   **Goal:** Establish immediate philosophical weight.

### Section 2: The Architecture (The "Strategic-Triangle")
*   **Visual:** Interactive 3D model or Diagram.
*   **Concept:** Explain the 3 pillars clearly.
    *   **Brain:** Intent (Why)
    *   **Body:** Execution (How)
    *   **Soul:** Ethics (For Whom)
*   **Interaction:** Hovering over one pillar dims the others, highlighting specific technologies (e.g., Brain -> LLMs, Body -> Blockchain).

### Section 3: The Call to Build (CTA)
*   **Not:** "Contact Us"
*   **Yes:** "Initialize Protocol" or "Read Whitepaper".
*   **Vibe:** Invitation to a movement, not a sales funnel.

## 3. Implementation Notes (React/Framer Motion)
*   Use `useScroll` to trigger text reveals.
*   Backgrounds should be subtle CSS gradients or WebGL, never static images.
