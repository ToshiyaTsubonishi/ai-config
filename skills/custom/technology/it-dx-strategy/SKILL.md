# it-dx-strategy (DX Commander)

## 1. Overview
「ITの力でビジネスを変える」ための戦略立案・実行スキルです。
技術的負債の解消と、新しいデジタルツールの導入を両輪で回し、組織全体のデジタル成熟度を高めます。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| DX成熟度診断・ロードマップ | **DX Maturity Auditor** | 組織のデジタル化レベルを定量的に評価し、次のステップを提示。 |
| 業務プロセス自動化・RPA | **Process Automator** | 定型業務をロボットに代行させ、人間を創造的業務にシフト。 |
| レガシー移行設計 | **Strangler Architect** | 巨大なモノリスシステムを、段階的にマイクロサービスへ移行。 |

## 3. Workflow
1. **Audit**: `DX Maturity Auditor` が各部署のIT活用状況を調査。
2. **Design**: `Strangler Architect` が、「2025年の崖」を回避するためのシステム刷新計画を作成。
3.  **Deploy**: `Process Automator` が、現場主導でRPAやNoCodeツールを導入できるよう支援。

## 4. Operational Principles
*   **User First**: システムの都合ではなく、ユーザー（社員・顧客）の利便性を最優先する。
*   **Agility**: 完璧な計画よりも、小さく始めて改善を繰り返す。
