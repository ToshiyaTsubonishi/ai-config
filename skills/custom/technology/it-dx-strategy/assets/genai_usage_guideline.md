# 生成AI利用ガイドライン (Generative AI Usage Guideline)

**制定日:** 202_/__/__
**対象:** 全役職員

## 1. 基本原則
生成AI（ChatGPT等）は業務効率化の強力なツールですが、誤った利用は情報漏洩や権利侵害のリスクを招きます。本ガイドラインを遵守し、正しく活用してください。

## 2. 入力データの制限 (Input Rules)
以下の情報は、**絶対に入力してはいけません**（学習データとして再利用される恐れがあるため）。
*   **個人情報:** 顧客氏名、住所、電話番号、マイナンバー等。
*   **機密情報:** 未公開の決算データ、M&A情報、未発表製品のスペック、ソースコード（OSSを除く）。
*   **他者の著作物:** 許諾を得ていない文章やプログラム。

## 3. 出力結果の利用 (Output Rules)
*   **正確性の確認:** AIの回答には誤り（ハルシネーション）が含まれる可能性があります。必ず人間がファクトチェックを行ってください。
*   **権利侵害の確認:** 生成物が他者の著作権を侵害していないか注意してください。

## 4. 利用環境
会社が認可した以下のツールのみを使用してください。個人の無料アカウントでの業務利用は禁止します。
*   [ ] ChatGPT Enterprise
*   [ ] Microsoft Copilot
