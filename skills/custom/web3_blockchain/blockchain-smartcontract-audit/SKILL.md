# blockchain-smartcontract-audit (Security Shield)

## 1. Overview
「金融Innovator」として信頼できるWeb3環境を構築するための、コードセキュリティ特化スキルです。
DeFi、NFT、および決済基盤となるスマートコントラクトに対し、ハッキングリスクを未然に防ぐ「最後の砦」として機能します。

## 2. Capabilities & Routing
| Intent | Agent | Strategy |
| :--- | :--- | :--- |
| スマートコントラクト自動監査 | **Audit Report Generator** | SlitherやMythril等のツール結果を集約し、エグゼクティブ要約を作成。 |
| バグ報奨金プログラム運用 | **Bug Bounty Coordinator** | 外部のホワイトハッカーと連携し、発見された脆弱性をトリアージ。 |

## 3. Workflow
1. **Static Analysis**: `static-analysis-bot` による網羅的な脆弱性スキャン。
2. **Fuzzing**: `symbolic-fuzzer-agent` を用いた境界値や不変条件のランダムテスト。
3. **Formal Verification**: 数学的証明によるロジックの正当性確認。

## 4. Operational Principles
* **Zero Tolerance**: 重大な脆弱性（Critical/High）がある状態でのデプロイは絶対に許可しない。
* **Provenance**: 監査結果をオンチェーンに記録し、信頼性を可視化。
