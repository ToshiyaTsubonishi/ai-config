# DAO立ち上げチェックリスト (DAO Launch Checklist)

## Phase 1: 準備 (Preparation)
- [ ] **ミッション定義:** DAOの目的（Purpose）と価値観（Values）は明確か？
- [ ] **初期メンバー:** コアチームと初期の貢献者（Whales, Devs）は確保できているか？
- [ ] **コミュニケーション:** DiscordサーバーとForum（Discourse等）はセットアップ済みか？
- [ ] **法的整理:** DAOの法的実体（LLC, Foundation等）または法的な位置づけはクリアか？（弁護士確認）

## Phase 2: トークン & ガバナンス (Token & Governance)
- [ ] **トークン設計:** 総発行量、配分、ロックアップ期間（Vesting）は決定したか？
- [ ] **スマートコントラクト:** トークンおよびガバナンスコントラクト（Governor）は監査済みか？
- [ ] **スナップショット:** Snapshotスペースを作成し、投票戦略（1 token = 1 vote 等）を設定したか？
- [ ] **トレジャリー:** Gnosis Safeを作成し、初期の署名者（Signers）を設定したか？

## Phase 3: ローンチ (Launch)
- [ ] **アナウンス:** ホワイトペーパーまたはLitepaperを公開したか？
- [ ] **エアドロップ:** 対象者のスナップショットを取得し、配布（またはClaim）を開始したか？
- [ ] **初期提案:** 最初のガバナンス提案（憲章の制定など）を投稿したか？

## Phase 4: 運用 (Operation)
- [ ] **モデレーション:** コミュニティモデレーターを配置し、ルールの運用を開始したか？
- [ ] **委員会 (Committee):** 特定の領域（助成金、開発）を担当するワーキンググループを組成したか？
