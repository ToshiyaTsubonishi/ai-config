# Art Loan Workflow Design (Lite)

MCPを使わずに、美術品担保融資業務の運用設計を文章ベースで作るための軽量スキルです。

## Inputs

- ヒアリング内容（顧客種別、担保作品、希望額、返済条件）
- 制約（社内規程、審査基準、SLA、エスカレーション条件）

## Output

以下を必ず順に提示します。

1. 業務フェーズ一覧（受付 → 審査 → 契約 → 実行 → 返済管理 → クローズ）
2. 判定分岐（承認/保留/謝絶）と判断条件
3. 役割分担（営業・審査・法務・業務部）
4. 例外時フロー（未入金、期限超過、担保毀損、解除依頼）
5. 最低限の運用KPI（処理時間、承認率、延滞率）

## Guardrails

- 法務・コンプライアンス判断は確定せず「要確認」と明示する
- 数値前提が不足する場合は仮定を明記する
- 実装コードではなく、運用フロー設計に集中する
